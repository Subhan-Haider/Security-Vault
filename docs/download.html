📋 Installation Guide

3 steps to
get started.

1

Download the APK

Tap "Download APK" above. Your browser will download app-debug.apk to your Downloads folder. The file is approximately 8 MB.

2

Allow Unknown Sources

Go to Settings → Security → Install Unknown Apps. Select your file manager or browser and enable "Allow from this source". This only needs to be done once.

3

Install & Launch

Open your Downloads folder, tap app-debug.apk, and tap Install. Launch the app — it opens as a calculator. Set your secret PIN in the settings, then use it to unlock the vault.
