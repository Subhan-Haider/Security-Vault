❓ FAQ

Common questions.

Is my data ever uploaded to a server? +
No. Stealth Vault is 100% offline. All files and data remain on your device only. No accounts, no cloud sync, no telemetry of any kind.
What happens if I forget my PIN? +
For security reasons, there is no PIN recovery mechanism. This is intentional — if someone could recover your PIN, so could an attacker. Keep your PIN safe.
Can I use Stealth Vault on multiple devices? +
Currently, Stealth Vault is a single-device app. Vault data is encrypted with keys tied to your specific device's Android KeyStore and cannot be transferred automatically.
What encryption does Stealth Vault use? +
AES-256-GCM (Galois/Counter Mode) with authentication tags. Encryption keys are generated and stored inside the Android Keystore (Hardware Security Module), never exposed to the app's memory.
Does the App Locker work on all Android apps? +
The App Locker works with most standard Android apps. It uses the Accessibility Service API to detect foreground app changes. Some system apps or launchers may behave differently.
