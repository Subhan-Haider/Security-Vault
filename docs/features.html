🔐 Feature 02

AES-256-GCM Hardware Encryption

Every file entering the vault is encrypted immediately using AES-256-GCM — the same standard used by governments and financial institutions. Your encryption keys never leave the device's secure hardware.

  • AES-256 bit keys with Galois/Counter Mode (authenticated encryption)
  • Keys generated and stored inside Android KeyStore (Hardware Security Module)
  • Keys are never exposed to app memory — decryption happens at hardware level
  • Files stripped from Android Gallery the moment they enter the vault
  • Hidden in the app's private directory — inaccessible without root
  • Room DB metadata protected with SQLCipher database encryption

Active Anti-Intruder Protocols
