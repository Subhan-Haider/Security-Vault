## вңЁ Overview

**Stealth Vault** is a multi-platform privacy ecosystem. On the surface, it's an elegant calculator app. Underneath, it's a hardened encrypted sandbox for your most sensitive files, photos, videos, and app data.

This monorepo contains **two applications**:

| Platform | Directory | Description |
|:---:|:---:|:---|
| рҹ“ұ **Android** | `/app` | Native Kotlin app with hardware-backed AES-256 encryption, intruder detection, app locking, and a decoy PIN system |
| рҹ’» **Web** | `/web` | React + TypeScript companion вҖ” same decoy calculator UI, auto-lock, glassmorphism design, and Framer Motion animations |

### рҹ”җ &nbsp; Hardware-Grade Encryption

> Your files are invisible to the OS and encrypted at rest.

```
User PIN  в”Җв”Җв–ә  Android KeyStore (HSM)  в”Җв”Җв–ә  AES-256-GCM Key Derivation
                                                      в”Ӯ
                                                      в–ј
                               Encrypted Blob  в—„в”Җв”Җ  File / Photo / Video
                               (stored in app private dir вҖ” invisible to file explorers)
```

- рҹ”‘ Keys are sealed in **Android KeyStore** вҖ” never exposed to memory unless actively decrypting
- рҹ—„пёҸ Vault metadata stored in **Room DB + SQLCipher** (database-level encryption)
- рҹҡ« Vault files are **stripped from Android Gallery** and all OS scanners immediately on import

- When a locked app is opened by an intruder, the screen is replaced with a **convincing "Application has stopped" crash dialog**

## рҹҡҖ Getting Started

### рҹ“ұ Android

**Requirements:** Android Studio Jellyfish+, JDK 17, Android SDK 34+

```bash
# 1. Clone the repo
git clone https://github.com/Subhan-Haider/Security-Vault.git

# 2. Open in Android Studio вҶ’ File > Open вҶ’ select the repo root
# 3. Wait for Gradle sync to complete

# 4. Build the debug APK
./gradlew assembleDebug

# Output вҶ’ app/build/outputs/apk/debug/app-debug.apk
```
